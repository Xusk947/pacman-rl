- [ ] ASCII карта парсится и валидируется по легенде; ошибки содержат контекст (строка/символ/причина)
- [ ] Среда поддерживает 5 агентов (Pacman + 4 призрака), корректные правила стен/пеллет/коллизий и условия завершения
- [ ] Синие большие пилюли `o` работают: включают frightened на T шагов, меняют правила коллизий и респавнят призраков
- [ ] Наблюдения выдаются как full-grid one-hot `C x H x W` и различают роль агента через agent_id
- [ ] Одна shared модель используется для всех агентов (общие веса), и PPO обновляет её без расхождений device
- [ ] CUDA путь работает на `torch.cuda.is_available()` (модель/батчи на GPU), и есть CPU fallback
- [ ] SQLite схема имеет PK/NOT NULL/индексы; метрики пишутся короткими транзакциями и читаются обратно
- [ ] Telegram отчёты не спамят: редактирование каждые 100 шагов в течение 1 часа, затем новое сообщение
- [ ] Каждые ~2500 шагов отправляется файл SQLite (или корректно эмулируется в dry-run)
- [ ] CLI покрывает ключевые параметры (map, steps, seed, db path, telegram on/off) и не требует хранить секреты в коде
